# 一秒翻盘 / Flip Second

一个面向**微信内竖屏试玩**的 H5 动作爽游原型，主打：

- **单指拖拽瞄准，松手冲刺**
- **残血触发“逆转时刻”**
- **过关三选一强化，形成 Build 差异**
- **第 3 关 Boss 战，验证完整循环**
- **适合移动端 Demo、提案演示、快速试玩**

## 当前版本亮点

- 更完整的品牌包装与首屏表达
- 更清晰的 HUD 与状态提示
- 更强的击中 / 击杀 / 逆转反馈
- 移动端更友好的信息层级
- GitHub Pages 自动部署已接通

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
